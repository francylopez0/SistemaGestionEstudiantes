Este proyecto implementa un sistema de gestión de estudiantes utilizando listas enlazadas en C#.

Permite:

Agregar estudiantes
Listar estudiantes
Buscar estudiantes
Eliminar estudiantes
Gestionar materias por estudiante
Calcular el promedio académico
Funcionalidades principales
Uso de estructuras de datos (lista enlazada)
Gestión dinámica de materias
Modificación de notas
Eliminación de materias
Selección por índice para evitar errores de entrada
Tecnologías utilizadas
C#
.NET
Programación orientada a objetos
Ejecución
Abrir el proyecto en Visual Studio Code o Visual Studio
Ejecutar el programa
Seguir el menú en consola
